# happy_web_app
A lightweight Streamlit web application that allows users to explore correlations between economic factors and well-being using data from the World Happiness Report.
## Features

    Dynamic Axis Selection: Choose between GDP, Happiness Scores, and Generosity for both X and Y axes.

    Interactive Scatter Plots: Built with Plotly Express for zooming, panning, and hovering over data points.

    Real-time Updates: The chart refreshes instantly as soon as a user changes the selection in the dropdown menus.
